## What's here

This repository hosts my personal portfolio — a single, hand-built page (no template, no framework) presenting my work, my approach, and the mission behind it.

```
.
├── index.html      # the site
├── fonts/          # self-hosted Mona Sans (woff2)
└── README.md
```

Built with plain **HTML & CSS**, self-hosted **Mona Sans** for body type, and **Fraunces** + **Bricolage Grotesque** for display. No build step — it deploys straight to GitHub Pages.
